# METRIC WebGL Runtime

This directory is editable METRIC runtime code for browser-side 3D rendering.
 If a METRIC visual needs renderer, material,
postprocess, picking, camera, or depth-of-field behavior, change the runtime
here instead of building wrappers around it.

Some files are derived from local renderer-derived source and keep the required MIT license
attribution in `LICENSE` and `visual/NOTICE.md`.
