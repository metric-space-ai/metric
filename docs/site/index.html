Concept

Records first. Metrics make them computable.

METRIC starts from finite records and true pair metrics. Coordinate views, graphs, fields and dynamics are derived from that finite relation structure.

Records

Images, strings, curves, distributions, graphs, vectors and structured rows enter as finite records.

Metric

A true metric assigns one pair value. In applications this can be read as minimal recoding effort between records.

Space

Records plus metric become a finite space for search, entropy, correlation, mapping, monitoring and dynamics.

Browser view

The browser renders exported data. It does not recompute METRIC algorithms or invent missing results.
